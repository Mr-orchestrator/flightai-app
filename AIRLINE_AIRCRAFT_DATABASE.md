## How It Works

### In Code:
```python
# Airline mapping
carrier_code = "VJ"
airline_name = get_airline_name(carrier_code)
# Returns: "VietJet Air"

# Aircraft mapping
aircraft_code = "330"
aircraft_name = get_aircraft_name(aircraft_code)
# Returns: "Airbus A330"
```

### In Display:
The system automatically:
1. Receives IATA codes from Amadeus API
2. Looks up in local database
3. Displays full name if found
4. Falls back to code if not found
